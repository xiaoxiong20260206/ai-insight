ChatGPT File Library上线：文件跨会话永久保存
OpenAI
OpenAI于3月23日正式推出File Library功能。用户上传或创建的文件（PDF、表格、图片等）自动保存至Library，可跨会话复用。支持Composer中快速引用近期文件，可在侧边栏Library标签浏览全部上传记录，还能直接询问ChatGPT关于历史上传文件的问题。当前向Plus/Pro/Business用户滚动推出，European Economic Area稍后跟进。
NEW
Anthropic发布81,000人AI调研报告：人们真正用AI做什么？
Anthropic
Anthropic邀请81,000名Claude.ai用户分享AI使用方式、期望与顾虑，形成大规模定性研究。报告揭示当前AI使用的真实场景分布、用户最担忧的AI风险类型、以及AI辅助工作与替代工作的边界感知。是目前规模最大的AI用户行为一手调研。

Apple WWDC 2026定档6月8-12日，聚焦「AI advancements」
TechCrunch
苹果3月23日官宣WWDC 2026于6月8-12日在库比蒂诺举行，主题明确聚焦「AI advancements」。预期亮点：① 配备更强个人上下文+屏幕感知能力的新版Siri（或联动Gemini）；② Foundation Model框架重大升级；③ Xcode agentic coding工具（已接入Claude Agent + OpenAI Codex）。去年WWDC主打「Liquid Glass」设计几乎不提AI，今年将完全转向。
NEW
Codex学生计划：美加高校学生可领$100 AI编程额度
OpenAI
OpenAI于3月20日推出Codex for Students，面向美国和加拿大在校本科生。学生通过SheerID用大学邮箱完成验证后，可领取$100 ChatGPT credits（即2,500额度），用于扩展ChatGPT Free/Go/Plus/Pro计划之外的Codex使用量，12个月内有效。这是OpenAI扩大AI编程工具教育渗透的重要一步。

AI编程工具三强混战：Kiro崛起，Claude Code vs Cursor vs Copilot
Reddit/社区讨论

GPT-5.4 mini向免费用户开放，成「Thinking」功能入口
OpenAI
OpenAI在3月18日开始向ChatGPT Free/Go用户推出GPT-5.4 mini，通过Composer「+」菜单的「Thinking」功能使用。对Plus/Pro付费用户，GPT-5.4 mini作为高峰期限速后的备用推理模型，保障推理功能持续可用。GPT-5 Thinking mini将在30天内从模型选择器退出。
NEW
Claude 3月用量翻倍促销持续至3月28日
Anthropic
Anthropic的Claude 3月促销（3月13日-28日）仍在持续：每日非高峰时段（北京时间22:00-次日14:00）用量额度翻倍，即5小时使用量变为10小时。此举旨在提高非高峰时段算力利用率，同时为用户提供更充裕的使用空间。距离活动结束还有4天，重度用户可抓紧利用。

Anthropic投资$1亿建立Claude Partner Network
Anthropic
Anthropic宣布投资1亿美元建立Claude Partner Network，系统性地将Claude能力整合到企业生态合作伙伴中。这标志着Anthropic从单一模型提供商向AI平台生态运营商的战略转型，与OpenAI GPT企业合作体系形成直接竞争。
