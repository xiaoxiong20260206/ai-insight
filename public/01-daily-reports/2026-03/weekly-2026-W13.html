1 Sora关闭：AI视频赛道首次重大商业退场 📅 3/26日报 →
SF Chronicle · 3月25日 | 🔴 AI视频商业化里程碑
OpenAI正式关闭Sora AI视频应用，Disney同步终止1亿美元合作协议。Sora是6个月前以TikTok竞争者姿态上线的生成式视频产品。这不是技术上的失败，而是"商业逻辑不成立"的失败——AI变现的主战场正在从C端转向B端基础设施。
2 MiniMax M2.7：108天四代迭代，国产第一全球第四 📅 3/23日报 →
新浪新闻/MiniMax官方 · 3月22日 | 🔴 国产模型历史性突破
MiniMax M2.7发布仅5天，在PinchBench斩获国产第一/全球第四。从M2到M2.7共108天四代迭代，SWE-Pro 56.22%，成本仅为Claude Sonnet的8%。M2.5上线一周API调用量3.07万亿Token，登顶全球开源模型。
3 Cursor三连发：降价86%+常驻Agent+企业自托管GA 📅 3/28日报 →
Cursor官方/TechCrunch · 3月19-28日 | 🔴 AI编程2.0平台化
Cursor发布Composer 2（底层基于Kimi K2.5，降价86%）→ Automations（Slack/GitHub触发常驻Agent）→ Self-hosted Cloud Agents GA（企业代码不出网络）。Cursor年化收入超$2B，估值$29.3B，正从IDE插件进化为开发者OS。
4 腾讯AI Lab解散并入混元：大厂AI整合时代到来 📅 3/23日报 →
微信/科技媒体 · 3月22日 | 🔴 大厂AI组织整合标志性事件
腾讯AI Lab正式宣告解散，研究团队和资源整体并入混元团队。继百度、阿里之后，国内大厂AI的"研究vs产品"博弈宣告产品侧获胜。与月之暗面4倍估值增长对照：大厂走整合路线，独立公司走速度路线。
5 Kimi港股IPO筹备：中国AI应用商业化的价值验证时刻 📅 3/28日报 →
Bloomberg/彭博社 · 3月26日 | 🔴 中国AI商业化里程碑
月之暗面(Moonshot AI)已与中金公司和高盛展开初步IPO谈判，拟赴港上市，估值约$180亿美元。Kimi月活已超1亿，订阅量较去年同期暴增8280%，成为中国AI独角兽港股上市潮的最新重要成员。
🔥 周度洞察
1 AI消费产品退潮，基础设施加速扩张

Sora关闭（消费端退场）与Arm发布首款AGI CPU（基础设施扩张）同日发生，并非偶然。同周：美国3月融资骤降至$130亿（较2月大幅下滑，AI超大轮次减少），但中国大模型周调用量4.69万亿Token持续增长。AI行业正在进入"从C端泡沫回归B端基础设施"的分化阶段。

3/25 Sora宣告关闭，Disney终止$1亿合作
3/25 Arm AGI CPU发布，Meta首发客户
3/26 美国融资数据骤降至$130亿
3/28 中国大模型周调用量4.69万亿Token披露
趋势判断
能证明商业可行性的少数公司估值飙升，纯消费级AI独立App遭遇淘汰。AI变现的主战场正在从C端走向B端基础设施——Sora关闭与Arm进军数据中心同日，是行业重心迁移的清晰信号。
2 AI Coding工具战争进入平台化阶段：「谁建生态」决定终局

本周AI编程工具连续三层进化：降本（Composer 2降价86%）→ 常驻化（Automations+Auto Mode）→ 企业化（自托管GA）。Figma MCP开放Agent写权限，是AI编程渗透产品设计全流程的里程碑。Kimi K2.5被确认为Cursor底座，侧面证明了国产模型已进入全球一流水平。

3/23 Cursor Composer 2发布，降价86%
3/25 Cursor Self-hosted Cloud Agents GA
3/26 Claude Code Auto Mode发布
3/29 Figma MCP Server Beta，AI获设计文件写权限
趋势判断
AI编程工具进入2.0平台化阶段：竞争终局不是"谁写代码更好"，而是"谁拥有更完整的开发者工作流"。7天内四个关键节点，企业级市场是下一个战场。
3 中国AI两条路同步得到验证：大厂整合 vs 独立高速迭代

同一周内，腾讯AI Lab解散并入混元（大厂整合）、MiniMax M2.7全球第四（独立迭代）、月之暗面估值涨4倍冲$10亿D轮（独立崛起）三件事同时发生，形成完整的市场实验对照。国产大模型连续三周在OpenRouter调用量超越美国，前四名全为国产。

中国AI生态的多元化格局正在形成——大厂走品牌整合路线，独立公司走速度路线，两条路在当前阶段都被市场验证有效。
