## 🧠 大模型

### 白宫拟推AI模型发布前审查，监管风向突变

据NYT和Reuters报道，特朗普政府正讨论通过行政命令建立AI工作组，探索新模型发布前的政府审查流程。此举直接由Anthropic Mythos的网络攻击能力引发，安全专家警告该模型可能显著提升代码级漏洞发现的速度和复杂度。白宫官员称相关讨论为"推测"，但方向与此前撤销拜登AI安全令形成鲜明反转。AI安全团体同时向欧盟委员会指出其AI办公室缺乏Mythos访问权限和评估能力。

> **深度聚焦**：从"放开搞"到"先审后放"——AI监管的逻辑拐点
>
> 白宫审查动议的深层逻辑不是安全理想主义回归，而是能力事实倒逼。当AI模型可以自主发现并利用代码漏洞时，传统"先发布后治理"框架已经失灵。这标志着AI监管从伦理风险转向能力风险的范式转换。英国AI安全研究所已在Mythos发布一周内完成技术分析，而欧盟AI办公室承认缺乏评估能力——监管能力的差距正在成为新的地缘政治变量。

### Centaur模型发布：用1000万人类决策训练AI模拟人类推理

IBM与Helmholtz Munich联合发布Centaur AI模型，基于Meta Llama 3.1训练，使用来自160项心理学研究的超1000万个人类决策数据。该模型能处理逻辑问题、道德困境和日常选择，旨在揭示人类思维模式。

> **深度聚焦**：当AI开始"读心"——心理学驱动的大模型新范式
>
> Centaur代表了"像人"而非"比人强"的新竞争维度。心理学对齐路径如果成功，可能催生一批专注模拟人类特定认知模式的专用模型——对产品设计、用户研究、行为预测等领域有直接应用价值。

### GLM-5.1发布：国产开源模型编程能力首超Claude Opus 4.6

智谱AI发布GLM-5.1开源模型，SWE-Bench Pro得分58.4，首次以国产开源身份超越Claude Opus 4.6（57.3分）。支持8小时持续自主工作，发布当天腾讯、百度、字节等Day0适配，国产芯片集体完成推理适配。智谱同步提价10%，年内累计提价83%，但调用量反增400%。

> **深度聚焦**：偏科生的突围——GLM-5.1登顶背后的代价
>
> 编程场景的突破真实但代价明显——医疗掉24名、法律掉6名。同一天Anthropic推出Mythos Preview在SWE-Bench Pro拿到77.8分，领先近20分。但提价策略值得关注——年内提价83%调用量反增400%，证明中国企业用户对能力溢价有一定接受度。

## ⌨️ AI Coding

### AI编程信任危机：96%开发者不信任AI代码，仅48%始终检查

Sonar发布2026 State of Code Developer Survey（1149名开发者）。数据显示：96%开发者不完全信任AI生成代码功能正确性，但仅48%表示始终检查AI辅助代码。AI已占2026年提交代码的42%，预计2027年达65%。53%的开发者认为AI增加了技术债务——代码看起来正确但实际不可靠。

> **深度聚焦**：验证缺口——AI编程的暗面
>
> 当42%的提交代码来自AI而验证覆盖率不足一半时，技术债务积累速度将远超传统开发模式。更危险的是初级开发者生产力提升最大但审查能力最弱，形成恶性循环。这直接催生了AI Code Assurance新赛道，Sonar已率先布局。对AI IDE厂商而言，内建验证能力将从加分项变为必选项。

### xAI发布Grok 4.3：降价40%-60%抢市场，但基准仍落后

xAI于5月1日发布Grok 4.3，API输入价格降至$1.25/M tokens（降约40%），输出$2.50/M tokens（降约60%）。支持百万token上下文、原生视频输入。但Artificial Analysis Intelligence Index得分53，仍落后OpenAI和Anthropic最新版本。10位联合创始人已全部离职。

> **深度聚焦**：激进降价是竞争力不足的信号而非优势
