# AI-Insight 项目与 Skill 的关系

> **说明**: 本文档明确 AI-Insight 项目与相关 Skill 之间的职责边界和协作关系

---

## 职责划分

### AI-Insight 项目 (知识库)

**定位**: 数据存储与知识沉淀

| 内容 | 路径 | 说明 |
|------|------|------|
| 日报产出 | `01-daily-reports/` | 所有日报归档 |
| 深度调研 | `02-deep-research/` | 所有调研文档 |
| 追踪体系 | `03-tracking-registry/` | 人物/公司/信息源清单 |
| 知识沉淀 | `04-knowledge-base/` | 概念/最佳实践/洞察 |
| 对外输出 | `05-outputs/` | 报告/演示/文章 |
| 模板 | `templates/` | 日报/调研模板 |

### daily-ai-report Skill (执行能力)

**定位**: 日报生成的执行指南

**路径**: `~/.codeflicker/skills/daily-ai-report/`

**职责**:
1. 定义日报生成的工作流程
2. 提供搜索策略和信息采集方法
3. 指导如何产出高质量日报
4. **使用 AI-Insight 的追踪体系作为信息源清单**

### industry-research Skill (执行能力)

**定位**: 深度调研的执行指南

**路径**: `~/.codeflicker/skills/industry-research/`

**职责**:
1. 定义调研的方法论和工作流程
2. 提供多维度分析框架
3. 指导如何产出高质量调研
4. **使用 AI-Insight 的模板和追踪体系**

## 协作模式

```
┌─────────────────────────────────────────────────────────────┐
│                    用户发起调研/日报任务                       │
└─────────────────────────────────────────────────────────────┘
                              │
                              ▼
┌─────────────────────────────────────────────────────────────┐
│                    Skill 提供执行指南                         │
│  - daily-ai-report: 日报生成流程                              │
│  - industry-research: 调研方法论                              │
└─────────────────────────────────────────────────────────────┘
                              │
              ┌───────────────┴───────────────┐
              ▼                               ▼
┌─────────────────────────┐     ┌─────────────────────────────┐
│  读取 AI-Insight 资源     │     │    产出内容                  │
│  - 追踪体系               │     │    - 日报/调研文档            │
│  - 模板                  │     │    - 存入 AI-Insight          │
│  - 历史内容参考           │     │                              │
└─────────────────────────┘     └─────────────────────────────┘
```

## 关键原则

### 1. 单一数据源 (Single Source of Truth)

- **追踪体系** (人物/公司/信息源) 统一维护在 `AI-Insight/03-tracking-registry/`
- Skill 中不再维护独立的 `source-registry.md`，改为引用 AI-Insight

### 2. Skill 专注流程

- Skill 只定义"如何做"
- 不存储数据，只引用 AI-Insight

### 3. 产出归档

- 所有产出存入 AI-Insight 对应目录
- 日报 → `01-daily-reports/`
- 调研 → `02-deep-research/`
