## 核心产品

### 模型系列
- **Command R / R+**: 企业级语言模型，RAG优化
- **Command A**: 最新的Agent优化模型
- **Embed**: 文本嵌入模型
- **Rerank**: 搜索重排模型
- **Tiny Aya**: 小型多语言模型

### 企业产品
- **North**: Agent平台，企业自动化
- **企业私有部署**: 支持本地/私有云部署
- **API**: 开发者接口
- **AWS Bedrock集成**: 通过Bedrock提供

## 战略分析

### 差异化："企业安全AI"
1. **Transformer血统**: 创始人是Transformer论文合著者
2. **企业专属**: 不做C端，只做B端
3. **安全优先**: 数据不出企业环境
4. **RAG专家**: 在检索增强生成方面业界领先
5. **加拿大优势**: 多伦多AI生态 + 地缘中立

### 核心竞争力
- **RAG能力**: Command R系列在RAG场景表现最佳
- **企业信任**: 金融等高合规行业的首选
- **部署灵活**: 私有云、本地、SaaS全覆盖
- **成本效率**: 比GPT-4便宜，企业性价比高
