## 趋势洞察

### 趋势一：模型竞争从"跑分"转向"体验+垂直能力"

**现象**: 
- GPT-5.4 强调电子表格/文档/演示文稿编辑能力（投行分析师任务87.3%准确率）
- GPT-5.3 Instant 强调"减少幻觉"和"减少不必要拒绝"

**本质**: 头部实验室开始为具体商业场景深度优化模型能力。

**启示**: 评估模型时，除了通用榜单，更应关注特定场景的表现。

### 趋势二：Agentic Engineering成为主流术语

**现象**: Simon Willison 启动 Agentic Engineering Patterns 项目，系统化区分：
- **Vibe Coding**: 完全不看代码，让AI自主编写
- **Agentic Engineering**: 专业工程师使用代理放大专业知识

**核心模式**:
1. Writing Code is Cheap Now（代码写作成本很低了）
2. Red/Green TDD（测试驱动复兴）
3. Linear Walkthroughs（线性代码走读）
4. Hoard Things You Know How to Do（囤积技能清单）

**本质**: AI编程从"新奇玩具"进入"专业方法论"阶段。

### 趋势三：AI安全与国家安全的首次正面碰撞

**现象**: Anthropic CEO公开拒绝国防部的"大规模国内监控"和"完全自主武器"要求。

**本质**: AI安全从技术概念上升为现实政治立场，AI公司需要在政府合作与价值观之间做出选择。

**风险**: 国防部威胁将Anthropic列为"供应链风险"，这可能影响其企业客户。

### 趋势四：资本"赢家通吃"效应空前明显

**现象**: 2月前三大融资（OpenAI+Anthropic+Waymo）占全球风投83%。

**本质**: 资本认为只有头部玩家有机会建成"AI时代的操作系统"。

**风险**: 私募市场与公开市场的背离值得关注（S&P 500软件行业指数下跌20%）。

### 趋势五：November 2025 Inflection Point得到多方验证

**现象**: 多位业界人士确认2025年11月是关键转折点：
- Paul Ford (NYT): "以前$350,000的工作，现在周末用$200/月Claude完成"
- Andrej Karpathy: "12月前coding agents基本不工作，12月后基本可以工作"
- Donald Knuth: Claude Opus 4.6解决了他研究数周的开放问题
- Jack Clark: 使用AI研究代理，徒步时让机器完成以前一周的工作

**本质**: 这不是渐进式改进，而是质变。

### 趋势六：AI Coding工具从"对话"向"可视化"进化

**现象**: Cursor 2.6 的 MCP Apps 支持在Agent对话中嵌入交互式UI（设计图、数据图表）。

**本质**: AI不仅能写代码，还能直接展示设计稿、数据分析，降低上下文切换成本。

**建议**: 评估MCP Apps能否将常用工具的查看流程集成到IDE中。

### 趋势七：Claw（爪子）成为新类别

**现象**: Andrej Karpathy 定义 "Claw" 为 OpenClaw 类 Agent 系统的统称。

**特点**:
- 运行在个人硬件上
- 通过消息协议通信
- 支持直接指令和计划任务
- 专属emoji：🦞

**意义**: Agent从云端向边缘设备扩展，个人AI助手进入新阶段。
