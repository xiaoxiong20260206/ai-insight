## 模板1：核心范式对比表（全文概览必用）

适用：文章的 00 全文概览章节，展示新旧范式的本质差异

```markdown
| 维度 | [旧范式名称] | [新范式名称] |
|------|-------------|-------------|
| 用户要什么 | [描述] | [描述] |
| 价值评判 | [描述] | [描述] |
| 失望后果 | [描述] | [描述] |
| 成本结构 | [描述] | [描述] |
| 传播方式 | [描述] | [描述] |
| 推广时机 | [描述] | [描述] |
```

**参考示例（SaaS vs AI）**：

| 维度 | SaaS（买工具） | AI产品（招员工） |
|------|----------------|-----------------|
| 用户要什么 | 能用这个功能 | 帮我把这件事做好 |
| 价值评判 | 有功能 = 有价值 | 结果好 = 才有价值 |
| 失望后果 | 换个功能继续用 | 离开+告诉别人"AI没用" |
| 成本结构 | 边际成本趋近零 | 每次推理都烧算力 |
| 传播方式 | 渠道驱动（广告/KOL） | 口碑驱动（爽感自发分享） |
| 推广时机 | 任何时候都可以 | 必须穿越质量阈值后 |

## 模板2：错误清单表（批判性分析章节）

适用：列举常见错误/反模式，每行一个

```markdown
| 错误 | 根因 | 后果 |
|------|------|------|
| **错误1：[标题]** | [根因] | [后果] |
| **错误2：[标题]** | [根因] | [后果] |
```

**变体：带示例**：

```markdown
| 错误 | 典型表现 | 为什么错了 |
|------|---------|---------|
| **堆功能** | 发布会展示30个功能，每个写"首创" | 质量不够的功能越多，负面体验面越大 |
```

## 模板3：产品五维评分表

适用：横向对比多个产品/解决方案

```markdown
| 产品 | Q 价值密度 | E 成本收支 | T 信任 | D 聚焦 | 结论 |
|------|-----------|-----------|--------|--------|------|
| [产品A] | ★★★★★ | ★★★★★ | ★★★★★ | ★★★★★ | ✅ 教科书 |
| [产品B] | ★★★★ | ★★★★ | ★★★★★ | ★★★★ | ✅ 方向正确 |
| [产品C] | ★★★★ | ★★ | ★★★ | ★★★ | ⚠️ 验证了需求 |
| [产品D] | ★★★ | ★★ | ★★ | ★★ | ⚠️ 通用陷阱 |
```

**评分说明**：
- ★★★★★：行业顶级，可作为标杆
- ★★★★：良好，方向正确
- ★★★：及格，还有明显短板
- ★★：问题较多，需要改进
- ★：严重缺陷

## 模板5：状态/阈值效应表

适用：描述某个临界点两侧的截然不同结果

```markdown
| 状态 | 路径 |
|------|------|
| 低于阈值 | [负循环描述，用→连接各环节] |
| 高于阈值 | [正循环描述，用→连接各环节] |
```

**示例**：

| 状态 | 路径 |
|------|------|
| 低于阈值 | 试用 → "不如自己来" → 不再用 → 告诉朋友"AI没用" |
| 高于阈值 | 试用 → "比我自己做得好！" → 形成习惯 → 告诉朋友"你一定要试" |

## 模板6：决策对比表（为什么这样做）

适用：解释一个反常识决策的原因

```markdown
| 理由 | 说明 |
|------|------|
| 理由1（简洁标题） | [一句话说明这个理由] |
| 理由2 | [一句话说明] |
| 理由3 | [一句话说明] |
```

**示例（为什么Day 1就要收费）**：

| 理由 | 说明 |
|------|------|
| 边际成本为正 | 每个用户每次用都烧算力，不收费就在亏钱 |
| 付费筛选优质反馈 | 付费用户的反馈精准，帮产品更快穿越阈值 |
| 锚点不可逆 | 先做免费，锚点定格为零，之后转收费极难 |

## 表格使用原则

1. **不少于3张**：整篇文章表格少于3张，内容可读性会大幅下降
2. **表头简洁**：每列表头不超过6个字，越短越好
3. **对齐方式**：正文表格默认左对齐，不需要手动设置（Markdown导入后手动居中）
4. **关键词加粗**：表格单元格中的关键词/数字用 `**加粗**`
5. **来源列标注**：引用外部数据/案例的表格要有来源标注
6. **禁止嵌套**：在 Markdown 源文件中不要嵌套表格（KIM Doc渲染可能出错）
