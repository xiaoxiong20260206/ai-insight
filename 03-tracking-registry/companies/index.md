## 分类体系

采用**多维度分类**，每个公司可同时属于多个维度：

| 维度 | 分类 | 说明 |
|------|------|------|
| **地区** | 海外 / 中国 | 公司总部所在地 |
| **类型** | 实验室 / 工具 / 应用 / 基础设施 | 核心业务类型 |
| **规模** | Tier1 / Tier2 / Tier3 | 行业影响力等级 |

## 追踪优先级调整规则

| 信号 | 动作 |
|------|------|
| 发布重大产品更新 | 提升检查频率 |
| 连续1个月无重要更新 | 降低检查频率 |
| 获得大额融资(>1亿美元) | 加强关注, 考虑升级Tier |
| 发布开源模型 | 测试评估, 加入追踪 |
| 核心人物离职 | 关注后续动态 |
| 出现重大负面新闻 | 风险标注 |
