## 🏆 本周 Top 5

> 从80+条资讯中精选最值得关注的5件事

### 1. GPT-5.4发布：首个原生Computer Use模型，OSWorld首超人类
- **来源**: OpenAI · 3月5日
- **摘要**: GPT-5.4是OpenAI首个具备原生计算机使用能力的通用模型，整合推理、编程和Agent工作流。OSWorld达75.0%（超越人类72.4%），支持1M token上下文。
- **为什么重要**: AI从"能聊天"跨越到"能操作电脑"。Computer Use是Agent落地最关键的基础设施升级——不需要API，只要有屏幕就能操作任何软件，对遗留系统意义重大。

### 2. 2月全球风投$1890亿创历史记录，83%流向三巨头
- **来源**: Crunchbase · 3月2-6日
- **摘要**: AI相关融资$1710亿占全球90%。OpenAI $1100亿（估值$8400亿）、Anthropic $300亿（估值$3800亿）、Waymo $160亿。美国占全球风投92%。种子轮反降11%。
- **为什么重要**: 资本极端集中+种子轮萎缩="冰火两重天"。AI创业门槛急剧提高，但差异化赛道（具身智能、物理AI）竞争相对较小。

### 3. Anthropic发布AI就业影响研究：白领大衰退"绝对可能"
- **来源**: Anthropic · 3月6日
- **摘要**: 引入"observed exposure"新指标。程序员75%任务被AI覆盖；22-25岁高曝光岗位入职率下降14%；理论覆盖94%但实际仅33%。同周Block裁员近半+美国非农减少9.2万人形成三重验证。
- **为什么重要**: AI就业讨论从"会不会影响"进入"正在如何影响"。最受冲击的不是蓝领而是高学历高收入白领。

### 4. Claude登顶App Store：品质逆袭ChatGPT
- **来源**: Mashable · 3月2-3日
- **摘要**: Claude首次超越ChatGPT登顶美国App Store免费总榜第一。免费用户自1月增长超60%。但随后遭遇"前所未有需求"导致近3小时宕机。
- **为什么重要**: AI应用竞争从"谁先上线"转向"谁体验更好"。用户用脚投票证明产品品质可以逆袭先发优势和品牌认知。

### 5. MCP生态爆发：Figma集成+Tool Search+设计↔代码双向打通
- **来源**: GitHub/OpenAI/Figma · 3月6-8日
- **摘要**: GitHub Copilot集成Figma MCP Server，AI编程工具可直接读取设计稿。GPT-5.4 Tool Search在36个MCP服务器下token用量减少47%。Figma反向推出Copilot to Figma，形成双向闭环。
- **为什么重要**: MCP正在成为AI工具互联的标准协议。"全链路AI开发"从理想变为工程可行。

## 🔥 周度洞察

> 比日报更高维度的跨日交叉分析

### 1. OpenAI的"闪电战"：一周双发重塑行业格局

本周OpenAI以罕见的节奏发布GPT-5.3 Instant（3/3）和GPT-5.4（3/5），两个模型定位完全不同：5.3 Instant聚焦"用户体验"（降幻觉26.8%、减少说教），5.4聚焦"Agent能力"（Computer Use超人类、Tool Search降成本47%）。

**趋势判断**: OpenAI正在以"闪电战"速度构建从基础模型到Agent工具链的完整生态护城河。这不是简单的产品迭代，而是从"对话AI"到"操作AI"的范式转变。

**支撑事件**:
- 3/3 GPT-5.3 Instant发布：首次以"用户体验"而非Benchmark为卖点
- 3/4 Codex桌面App登陆Windows：多Agent并行成标配
- 3/5 GPT-5.4发布：Computer Use + Tool Search + 1M上下文
- 3/6 GPT-5.4整合Codex编程能力：通用模型与编程专用模型合流
- 3/7 GPT-5.4安全评估报告发布：安全与能力并行

### 2. AI就业冲击三重验证：从理论到现实的关键一周

Anthropic研究（3/6）、Block裁员近半（3/6）、美国2月非农减少9.2万（3/6）三者在同一周汇聚，将AI就业讨论推入新阶段。但Salesforce CEO的"AI washing"质疑提醒我们区分"真AI驱动"和"借AI之名"。

**趋势判断**: 2026年将成为"AI就业冲击元年"。关键不在失业规模，而在就业结构变革——高学历高收入白领（程序员、分析师、律师）首当其冲。

**关键数据对比**:
- 理论覆盖94% vs 实际使用33% = 61%的鸿沟待收窄
- Computer Use能力正是弥合鸿沟的关键技术
- 预计2027-2028年实际覆盖率将升至55-65%

### 3. AI从"独立应用"到"嵌入式能力"的范式转变

ChatGPT进军Excel（3/6）、Figma双向集成（3/6）、Cursor MCP Apps（3/3）——AI不再是"打开一个App去用"，而是"在你已经使用的工具里等你"。

**趋势判断**: 2026年下半年，不集成AI的SaaS产品将面临用户流失。竞争从"谁有AI"转向"AI在哪里出现"。
